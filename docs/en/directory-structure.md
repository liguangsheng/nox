# Directory Structure

Top-level layout:

```text
.
├── Cargo.toml
├── Cargo.lock
├── README.md
├── README_zh_CN.md
├── crates/
├── docs/
├── examples/
├── tests/
└── target/
```

Key ownership rules:

- `README.md` is the English quick entry.
- `README_zh_CN.md` is the Chinese quick entry.
- `crates/nox_core` owns the embeddable engine.
- `crates/nox` owns the default runtime and CLI.
- `docs/en` contains English documentation.
- `docs/zh_CN` contains Chinese documentation and ADRs.
- `examples` contains runnable positive scripts, sample projects, and embedding smoke inputs.
- `tests` contains `.nox` fixtures, malformed corpora, and benchmark inputs used by automated checks.
- `target` is generated by Cargo and must not be committed.

The detailed Chinese directory guide is available in [`../zh_CN/directory-structure.md`](../zh_CN/directory-structure.md).
